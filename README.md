# Screen Dimmer

Screen Dimmer is a Windows desktop utility written with PySide6 that places a frameless color overlay on a display so you can dim the screen with adjustable opacity, custom tint, and configurable animations.

For the full technical breakdown, repository layout, runtime data, install flow, build flow, and internal flags, see [Specifications](Specifications.md).

## Main Features

- Frameless full-screen dim overlay
- Adjustable per-display opacity
- Custom per-display overlay color
- Compact color picker with hex input
- Per-display profile persistence
- Configurable fade-in and fade-out animation timing
- Configurable zoom-in and zoom-out animation timing and scale
- Adjustable animation frame rate
- Light and dark UI themes for the settings interface
- Single-instance behavior with IPC-based activation
- Custom frameless installer and uninstaller UI
- Editable Python workflow and packaged executable workflow in the same project
